Data Privacy Policy

We use several data processing technologies that collect personal data for specific purposes. As the data subject, you can manage your preference to limit the extent of your private data that this website can access.

For your better management, we are obliged to provide you the information on how and why we, as the data controller, utilize such technologies.

Web Storage

Web Storage API enables websites to store key/value pair data in a browser on a device by generating Storage objects. The Storage objects will reserve pairs of data 5MB maximum that the data controllers can handle as javascript objects. Your data will stay secure without any additional transmission with external servers until necessary data processing occurs. The data is retrievable when any script in the website requests it for use.

We classify the data in Web storage with three categorizations:

  • Purpose that the data serve
  • Duration for which the data stay
  • Provenance where the data have been processed

Duration

  • Session Storage keeps pairs of data until the session closes. The data will disappear when a user ceases the current session.
  • Local Storage preserves data across sessions. The data will last persistent after a browser has closed and reopened.

Purpose

  • Strictly Necessary Data cope with essential functions. For instance, the website needs to remember your permission for the usage of personal data. Hence, the associated key/value pair is necessary.
  • Functional Data allow a website to remember your choices, such as preferred language or graphic settings, to maintain your customized experience of the website functions.
  • Analytic Data gather information about users' overall activities and performances, including how you use this website. We analyze such anonymized data to develop and improve website functions.
  • Marketing Data track your online activity to help advertisers deliver more relevant advertising or limit how many times you see an ad. Other organizations or advertisers can monitor and share your data if you permit.

Provenance

  • First-party Data is put on your device directly by the website you are visiting.
  • Third-party Data is placed on your device, not by the website you are visiting, but by a third party like an advertiser or an analytic system.
